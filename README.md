# UAV-APM Website

This repository contains the source for the UAV-APM project website hosted at:

https://uav-apm.github.io/

The site is built with the Academic Pages / Jekyll template and provides documentation and public-facing information for the UAV-APM project, including project descriptions, documentation, downloads, tutorials, publications, and contact information.

Summary of what you'll find on the live site
- Home / news and highlights about UAV-APM
- About: project mission and contributors
- Projects: active UAV projects
- Publications and talks
- Contact and contribution information

Get started (quick)
1. Visit the live site: https://uav-apm.github.io/
2. Browse documentation and downloads from the Documentation and Downloads pages.
3. To propose changes or corrections, open a pull request against this repository.

Reporting issues
- Report bugs or request features by opening issues in this repository's Issues tab.
